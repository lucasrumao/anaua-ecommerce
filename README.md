## ✨ Visão Geral

O **Anauá** é um projeto de e-commerce desenvolvido com foco em **UI/UX moderno** e **simulação de regras reais de negócio**, inspirado em marcas de skincare com ativos brasileiros.

A proposta vai além de um layout estático: o sistema implementa interações dinâmicas e comportamentos reais de uma loja virtual.

🔗 **Acesse a demo:**  
https://lucasrumao.github.io/anaua-ecommerce

## 🚀 Funcionalidades

### 🛒 Carrinho Inteligente
- Adição e remoção de produtos em tempo real  
- Persistência com `localStorage`  
- Atualização dinâmica de valores  

### 📦 Controle de Estoque
- Produtos esgotados são identificados automaticamente  
- Interface adaptada (visual + bloqueio de ações)  

### 🚚 Cálculo de Frete
- Integração com backend em Node.js  
- Consulta de CEP para cálculo de envio  

### 🎯 Experiência do Usuário
- Carrinho lateral (sidebar)  
- Barra de progresso para frete grátis  
- Interface fluida e responsiva  

---

## 🛠️ Tecnologias

**Front-end**
- HTML5  
- CSS3 (Flexbox, Grid, variáveis CSS)  
- JavaScript (ES6+)  

**Back-end**
- Node.js  
- Express  

**Outros**
- LocalStorage  
- API de CEP

## 💻 Como rodar localmente

```bash
git clone https://github.com/lucasrumao/anaua-ecommerce.git
cd anaua-ecommerce 
```

### Backend (opcional)

```bash
npm install
node server.js
```
